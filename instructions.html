Welcome to the map generator!

Instructions on how to use the map generator:


The menu:

  • Width and height inputs affect the map width and height, only enter in integers greater than zero.
  • Information image when clicked on will pull up this menu if you need a refresher on how to use the page.
  • Download image button will save the map as an image to your computer.
  • Terrain buttons are currently not working and are being worked on.

The navigation compass:

  • Use the buttons to traverse the map, only works on larger maps that are bigger than your screen.
